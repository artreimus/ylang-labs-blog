# Prompt Starters

Use these only when the user asks for examples, wants the attached scientific blueprint style adapted, or needs a fast starting prompt.

## Planetary Atlas Starter

Style reference: `references/earth-atlas-blueprint.png`

```text
Create a white-on-cobalt scientific blueprint poster of [SUBJECT] as a planetary atlas sheet. Place one dominant central globe or technical object on deep blue graph paper, with crisp white linework, graticule guide lines, dimension arrows, sparse metadata panels, a border frame, and a compact title block. Use short generic drafting labels only; keep exact numbers out of the image unless supplied.
```

## Accelerator Map Starter

Style reference: `references/large-hadron-collider-blueprint.png`

```text
Create a white-on-cobalt scientific infrastructure blueprint board of [SUBJECT]. Show a large schematic ring or route map with labeled nodes, directional arrows, injector lines, leader lines, smaller inset maps, a cross-section diagram, and a lower-corner specification table. Use a deep blue graph-paper background, precise white drafting linework, framed borders, and sparse readable labels.
```

## AI Engineering Adaptations

- Agent loop: circular accelerator-style route with stations for observe, plan, act, verify, and memory.
- RAG pipeline: infrastructure board with retrieval, ranking, synthesis, citation, and evaluation nodes.
- Model-serving stack: atlas poster with central service core, inset latency/memory panels, and cross-section layers.
- Evaluation harness: engineering education chart with test suites, traces, metrics, and failure-mode callouts.
- Multi-agent system: topology poster with orbital worker nodes, shared context bus, memory vault, tool ports, and verification lanes.

Keep exact API names, metrics, and claims in the MDX caption or article body unless the user explicitly requires short labels inside the image.
